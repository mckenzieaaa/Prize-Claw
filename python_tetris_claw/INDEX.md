## 📞 获取帮助

### 常见问题

**Q: 游戏无法启动？**
A: 确保安装了 pygame：`pip3 install pygame`

**Q: 控制不响应？**
A: 点击游戏窗口确保其获得焦点

**Q: 如何退出游戏？**
A: 在主菜单按 ESC 键

**Q: 高分会保存吗？**
A: 当前版本仅保存在内存中，关闭游戏后会丢失

**Q: 如何添加新功能？**
A: 参考 [DEVELOPER_GUIDE.md](DEVELOPER_GUIDE.md) 的"添加新功能"部分
