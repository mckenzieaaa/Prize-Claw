## 🚀 运行游戏

### 安装
```bash
cd python_tetris_claw
pip3 install -r requirements.txt
```

### 运行
```bash
python3 main.py
```

### 控制
- **WASD** - 移动爪子
- **空格** - 抓取/确认
- **ESC** - 返回/退出
